# 项目名称:BattleChat
## 项目背景
 随着科技的发展，聊天交友软件层出不穷。近几年又出现了《元宇宙》、《Web3》等这些名词，使我有了做个两者相结合的聊天交友软件。
 ## 配置
 框架:React
 主要SDK:agora-rtc-sdk-ng4.13.0、easemob-websdk4.0.9
 ## 运行
 1.进入项目根目录，执行‘npm install’
 2.执行'npm start'，启动成功后，浏览器打开'http://localhost:3000/'
